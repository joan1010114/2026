# 無二電商事務所 - 銷售頁

線上版本：https://land.52ec.tw

## 檔案結構

```
.
├── index.html        # 主頁
├── .htaccess         # Apache 設定
└── images/           # 所有圖片素材
    ├── logo-52ec-black.png
    ├── logo-52ec-white.png
    ├── hero-fuhan.png
    ├── ad-*.png      # 4 張廣告圖
    ├── mockup-*.png  # 三平台 mockup
    ├── case-*.jpg    # 案例截圖
    └── clients/      # 12 個客戶 logo
```

## 部署

把所有檔案上傳到 Cloudways 的 `public_html/` 即可。

## 修改

直接編輯 `index.html` 文字內容,推到 GitHub 自動部署。
